前天刚把 Hermes 配好飞书通道，昨天就看到一个大消息——

**Hermes 原生支持个人微信了！**

**来源：雷神**

X 大神开发了一款适配器，用腾讯的 **iLink Bot API** 连接个人微信（不是企业微信）。消息走长轮询，不用公网 IP，不用配 Webhook，扫码就能连。

为什么我这么激动？

因为用了两天 Hermes 之后，我发现它比龙虾（OpenClaw）强太多了：

| 功能       | Hermes            | 龙虾           |
| ---------- | ----------------- | -------------- |
| 记忆管理   | ✅ 自动更新        | ❌ 手动维护     |
| Skill 创建 | ✅ 自动创建        | ❌ 手动配置     |
| 任务规划   | ✅ 能规划 80+ 步骤 | ⚠️ 步骤多了会崩 |
| Token 消耗 | ✅ 更省            | ❌ 更费         |

现在加上微信通道，真香预警。

## **一、 一句话升级**

##  

如果你已经部署过 Hermes，升级只需要一行命令：

```
hermes update
```

**操作步骤：**

1.在 Ubuntu 终端按 `Ctrl + C` 停止正在运行的 Hermes

2.输入 `hermes update`，回车

3.等待更新完成

**小惊喜**：Hermes 更新时如果检测到本地冲突，会自动帮你暂存。龙虾遇到冲突直接报错，得手动处理。

## **二、这次更新带了什么**

##  

更新完成后，系统会提示恢复本地更改，先选「手动恢复」，把微信配置完再说。

**本次更新内容：**

| 类型       | 内容                                                 |
| ---------- | ---------------------------------------------------- |
| ➕ 新增技能 | `ideation`（创意/头脑风暴）                          |
| ⬆️ 更新技能 | `manim-video`、`ascii-video`、`p5js`                 |
| ➖ 移除技能 | 9 个技能从官方清单移除（本地文件还在，只是不维护了） |

##  

## **三、配置 LLM Wiki（可选）**

##  

Hermes 原生自带 LLM Wiki 技能，配置一下目录就能用。

```
# 配置 wiki 目录（默认即可）
hermes wiki setup
```

##  

## **四、运行安装向导**

##  

微信配置需要走交互式安装向导：

```
hermes gateway setup
```

这个命令会启动网关配置流程，同时支持微信、飞书、企微，可以一起配。

## **五、微信扫码绑定**

##  

**步骤：**

1.选择「微信」，输入 `Y` 确认

2.终端会生成一个二维码链接（因为终端无法直接渲染二维码）

**3.复制链接 → 浏览器打开 → 微信扫码**

**重要提醒**：

如果你的微信已经绑定了其他 OpenClaw 产品（比如龙虾），扫码后会**自动解绑**

微信 Bot 同时只能对接一个 AI 助手

## **六、权限配置**

##  

扫码成功后，会进入权限配置：

**对话权限**：建议全部放开（除非你有特殊安全需求）

**群聊配置**：同样建议放开所有群

**主通道选择**：

选「微信」→ 所有回复走微信

选「飞书」→ 飞书消息卡片更美观（推荐）

配置完成输入 `Done` 退出。

## **七、重启测试**

##  

配置完成后重启网关：

```
hermes gateway restart
```

然后在微信里发一条消息测试，看到 Hermes 回复就成功了！

🔍**群聊支持**：官方说群聊也支持，但雷神研究了半天没搞定，可能是腾讯的限制，后续再探索。
